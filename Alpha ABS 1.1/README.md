### How install:
>You need RPG Maker MV 1.3.1 or above.  

Copy contents  of the [Plugin folder](https://github.com/KageDesu/Alpha-ABS/tree/master/Alpha%20ABS%201.1/Plugin) in your project.  
Чтобы установить плагин, скопируйте содержимое [папки Plugin](https://github.com/KageDesu/Alpha-ABS/tree/master/Alpha%20ABS%201.1/Plugin) в свой проект.

**Alpha ABS required only one plugin file: Alpha_ABS.js**, don't use other plugins from Demo game in your project, it may cause errors.

Video guide on [YouTube](https://www.youtube.com/watch?v=1wXL39bwx0g)
